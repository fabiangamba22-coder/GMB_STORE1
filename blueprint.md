# **App Name**: NeoDev Studio

## Core Features:

- AI Design System Extraction Tool: An AI tool to analyze visual inputs (like screenshots or style guides) and generate a structured JSON-based design system, including coherent colors, typography rules, spacing, and fundamental component definitions for UI development.
- AI UI Dashboard Generator: An AI tool that generates complete, single-page Next.js dashboard layouts, styled as card-based designs, based on user-defined requirements such as metrics, courses, and tasks, utilizing a chosen or AI-extracted design system.
- Dynamic Component Integration & Adaptation: Intelligently adapt and integrate complex UI components (e.g., 3D cards with hover effects, animated sections) into generated Next.js layouts, ensuring visual consistency with the active design system and correct functionality (e.g., installing framer-motion).
- Custom Style & Component Injection: Allow users to inject raw CSS code, custom React components, or specific styling directives (like glowing effects or highlight components for titles) to customize generated UI elements and layouts beyond standard parameters.
- Interactive UI Preview: Provide a real-time, interactive preview environment for generated Next.js UI, allowing developers to immediately visualize how layouts and components function and appear before exporting the code.
- Generated Asset Storage & Management: Securely store and manage all user-generated JSON design systems, UI code snippets (HTML, CSS, JS), and custom components within a Firebase-integrated database for easy retrieval and versioning.
- One-Click Code Export: Enable seamless download and export of the generated Next.js compatible HTML, CSS, and JavaScript files, ready for direct integration into development projects.

## Style Guidelines:

- Primary color: A vibrant cyan (#00F2FF) that commands attention and conveys a technological, modern aesthetic, suitable for calls to action and key elements in a dark environment.
- Background color: A very dark, desaturated teal-blue (#1A2426). This dark shade provides a subtle hue consistency with the primary color, creating depth while ensuring high contrast for foreground elements.
- Accent color: A vivid mint green (#B4FFEE). This lighter, high-saturation color is analogous to the primary and offers a striking contrast in brightness and saturation, ideal for secondary highlights, interactive states, or decorative elements, fitting the 'neon' concept.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly legible appearance across all text sizes, maintaining a consistent technical and professional feel.
- Use sleek, high-contrast, line-based icons that embody a 'cyber' and 'gamer' aesthetic. Icons should have subtle glowing effects or clear states to indicate interactivity without visual clutter.
- Prioritize a card-based layout structure for modularity and scalability, with clearly defined sections. Utilize dark, bordered cards to frame content effectively, contributing to the high-contrast dark theme.
- Implement subtle hover animations (e.g., slight scale increase, enhanced border glow) and smooth 0.3s ease-in-out transitions to create an engaging and responsive user experience, aligning with dynamic effects mentioned in the inspiration.
